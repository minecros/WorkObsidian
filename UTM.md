**Unified Threat Management (UTM)** is a comprehensive security solution that combines multiple network protection functions into a single, integrated platform, simplifying management.
**Like a Swiss army knife, the Unified Threat Management (UTM) provides multiple features in a single appliance.** Its features can include:
- Network Firewalling
- Network Intrusion Prevention (IPS)
- Gateway Antivirus (AV)
- Gateway Anti-Spam
- Virtual Private Network (VPN)
- Content Filtering
- Load Balancing
- Data Leak Prevention (DLP)
It is one-stop shopping.  You must consider that it is a single point of failure and may provide a lot more features than you really need.  However, it is better than zero protection.  It does require care and feeding.
